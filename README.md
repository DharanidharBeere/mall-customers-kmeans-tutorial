# How the Choice of k Affects k-Means Clustering Performance and Cluster Quality 

This repository contains all code, data, figures and the tutorial PDF for the machine learning tutorial:

**"How the Choice of k Affects k-Means Clustering Performance and Cluster Quality"**

## Contents
- Jupyter Notebook with full experimental analysis
- Tutorial PDF
- Mall Customer Dataset
- Generated figures
- Requirements file
- Open-source licence

## How to Run the Code

1. Install Python 3.9+  
2. Install dependencies:
   pip install -r requirements.txt  

3. Run the notebook:
   jupyter notebook notebook.ipynb

All figures used in the tutorial will be reproduced automatically.

## Dataset
The Mall Customer Segmentation dataset is used for clustering analysis.
